Temperatura
  • Para Grados Fahrenheirt, 1 centígrado es igual a 33.8 grados, para esto se ocupa la formula de ((centigrados * 1.8) + 32).
  • Para Grados Kelvin, 1 centígrado es igual a 273.15 grados, para esto se ocupa la formula de (centigrados + 273.15).
  • Para Grados Fahrenheirt a Centígrados, 1 Fahrenheirt es igual a -17.05 grados, para esto se ocupa la formula de ((tem - 32) * 0.55).
  • Para Grados Fahreheirt a Kelvin, 1 Fahrenheirt es igual a 291.29 grados, para esto se ocupa la formula de ((tem + 32) * 0.55 + 273.15).
  • Para Grados Kelvin a Centígrados, 1 Kelvin es igual a -272.15 grados, para esto se ocupa la formula de (tem - 273.15).
  • Para Grados Kelvin a Fahrenheirt, 1 Kelvin es igual a -457.86 grados, para esto se ocupa la formula de ((tem - 273.15) * 1.8 + 32).
Longitudes
  • Para las Pulgadas, 1 pulgada es igual a 2.54 centímetros, para esto se ocupa la formula de (centímetros / 2.54).
  • Para los Pies, 1 pie es igual a 30.48 centímetros, para esto se ocupa la formula de (centímetros / 30.48).
  • Para las Yardas, 1 Yarda es igual a 91.44 centímetros, para esto se ocupa la formula de (centímetros / 91.44).
  • Para los Millas, 1 Milla es igual a 1.609 centímetros, para esto se ocupa la formula de (centímetros / 1.609).
Monedas

  • Para el Euro, 1 euro es igual a 26.2 pesos mexicanos, para esto se ocupa la formula de (peso mexicano / 26.2).
  • Para la Libra Esterlina, 1 esterlina es igual a 30.05 pesos mexicanos, para esto se ocupa la formula de (peso mexicano / 30.05).
  • Para el Peso Chileno, 1 peso chileno es igual a 0.028 pesos mexicanos, para esto se ocupa la formula de (peso mexicano / 0.028).
  • Para el Peso Uruguayo, 1 peso uruguayo es igual a 0.56 pesos mexicanos, para esto se ocupa la formula de (peso mexicano / 0.56).
  • Para la Rupia, 1 rupio es igual a 0.32 pesos emxicanos, para esto se ocupa la formula de (peso mexicano / 0.32).
  • Para el Yen, 1 yen es igual a 0.22 pesos mexicanos, para esto se ocupa la formula de (peso mexicano / 0.22).
  • Para el Zloty, 1 zloty es igual a 5.78 pesos mexicanos, para esto se ocupa la formula de (peso mexicano / 5.78).
